# URL Shortener (Node.js)

This is a simple URL Shortener project built using Node.js, Express.js, and MongoDB.

## Features
- Convert long URL into short URL
- Redirect to original URL
- Simple and fast

## Technologies Used
- Node.js
- Express.js
- MongoDB

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node server.js`
4. Open browser at `http://localhost:3000`
